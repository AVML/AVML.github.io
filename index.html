About AVML

Use Audio Visual Markup Language (AVML) to create individual pages inside of a client app.

You control the flow of a client-server app through hypermedia.

Stream

Stream represents a audio/video stream.

  1. type Stream struct { ID string `json:"@id"` // The original source of the steam. For example if this stream is // the result of a transcoding process Canonical will provide an URI // to the source. This helps us use the source to resume transcoding // for example. Canonical string Index int `json:"index,omitempty"` Language string `json:"language,omitempty"` // Iso639_1 Label string `json:"label,omitempty"` Codec Codec VideoSpec // if codec_type is "video" AudioSpec // if codec_type is "audio" Disposition StreamDisposition `json:"disposition,omitempty"` Bitrate string `json:"bitrate,omitempty"` }

Series

Series represents a tv series.

  1. type Series struct { // the tv series parts/seasons Parts []SeriesPart } }

SeriesPart

SeriesPart represents a series part/season.

  1. type SeriesPart struct { // e.g. Episodes Parts []Media }

Header

Header represents media part header.

  1. type Header struct { ContentID string Name string Number int `json:",omitempty"` Description string `json:",omitempty"` Air time.Time // trailers/additional meta Related []MediaPart `json:",omitempty"` }

Media

Media represents a media container.

  1. type Media struct { // https://en.wikipedia.org/wiki/Breadcrumb_(navigation) Breadcrumbs []Header `json:",omitempty"` // content (e.g video files). The media content may be split // into multiple files (e.g CD1, CD2 ) even if it's basically // the same movie clip/episode etc Part MediaPart `json:",omitempty"` Meta }

Live

Live represents a live media stream.

  1. type Live struct { Media // the media here is actually the program // name (e.g. FOXNews) along with the current // media stream playing. IN the future we may // want to replace Media with a custom type Guide []Guide }

Guide

Guide represents the tv guide of a live program.

  1. type Guide struct { Start time.Time Stop time.Time Media }

Part represents a media part.

  1. type Part struct { mime.Mime Streams []Stream Format Format }

  1. type Meta struct { // genre is music for music files.? Genre []string `json:",omitempty"` Kind Kind Country []string `json:",omitempty"` Rating string `json:",omitempty"` Credits []Credit Release time.Time `json:",omitempty"` }
